About Me

I love interfaces, especially interactive ones. I specialize in forming the structures and actions on which web applications are built. From user flows and rapid prototypes to scripted web animations and css transitions, I'm here to make the web a prettier, easier place to live.

UI/UX Design

User Flows

Building user flows has been a large part of the success of the projects I've been a part of. User Stories help guide the project to useful conclusions and waste less time on empty features.

Wireframes

Wireframing is another tool constantly employed to minimize empty hours on development. Used to focus the content on a page, huge differences can be made on how the user processes information.

Prototypes

Making scalable prototypes is my favorite part of my job, whether it be a low fidelity prototype built in Axure or a full mock-up built for the web, there is nothing quite like creating something new and useful.

Front End Development

HTML

Clean, semantic markup is a constant and ever present demand to everything I build. Markup is the skeleton of any application, and done properly, can make a scaleable structure for future development.

CSS

Styling a web application is where I shine, I know the ins and outs of the styling cascade, specificity, animations and transitions. CSS makes the designs that make the web.

JavaScript

I use Javascript to breathe life into the designs I work on. JavaScript is one of the most valuable tools I have to bring action and motion to the websites and applications I build.
