About Me

I love interfaces, especially responsive ones. I specialize in forming the structures and actions on which web applications are built. From user flows and rapid prototypes to scripted web animations and css transitions, I'm here to make the web a prettier, easier place to live.

UI/UX Design

User Flows

Building user flows has been a large part of the success of the projects I've been a part of. I use stories and flows to help guide my projects and conserve time better spent on development.

Wireframes

Wireframing is another tool I employ to minimize waste on my projects. I use wireframes to focus the content on a page and map out user flows into traceable paths through applications.

Prototypes

Making scalable prototypes is my favorite part of my job, whether it be a low fidelity prototype built in Axure or a full mock-up built for the web, there is nothing quite like creating something new and useful.

Front End Development

HTML

Clean, semantic markup is an ever present requirement to everything I build. Markup is the skeleton of any application, and makes a scaleable structure for future development.

CSS

Styling a web application is where I shine, I know the ins and outs of the styling cascade, specificity, animations and transitions. I use CSS to unite design and function into the web we know and love.
