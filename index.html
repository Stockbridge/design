About Me

I love interfaces, especially responsive ones. I specialize in forming the structures and actions on which web applications are built. From user flows and rapid prototypes to scripted web animations and css transitions, I'm here to make the web a prettier, easier place to live.

Front End Development

HTML

Clean, semantic markup is an ever present requirement in everything I build. Markup is the skeleton of any application and makes a scaleable structure for future development.

CSS

Styling a web application is where I shine, I know the ins and outs of the styling cascade, specificity, animations and transitions. I use CSS to unite design and function into the web we know and love.

JavaScript

I use Javascript to breathe life into the designs I work on. JavaScript is one of the most valuable tools I have to bring action and motion to the websites and applications I build.

UI/UX Development

Atomic Design

Atomic design techniques prove to be some of my best tools in developing a scalable set of components in my projects, as well as reducing the overall weight and complexity of my code.

Design Systems

Creating a pattern lab for my projects is one of the ways I instill a coherent visual language and increase the speed of design and development.

Responsive Web Development

With the quickly escalating use of phones as a primary browser, it has become increasingly important for my websites to scale for all viewports. It can also be a great tool for keeping a site lean, fast, and user intuitive.

UI/UX Design

Prototypes

Making scalable prototypes is my favorite part of my job, whether it be a low fidelity prototype built in Axure or a full mock-up built for the web, there is nothing quite like creating something new and useful.

User Flows

Building user flows has been a large part of the success of the projects I've been a part of. I use stories and flows to help guide my projects and conserve time better spent on development.

Wireframes

Wireframing is another tool I employ to minimize waste on my projects. I use wireframes to focus the content on a page and map out user flows into traceable paths through applications.
